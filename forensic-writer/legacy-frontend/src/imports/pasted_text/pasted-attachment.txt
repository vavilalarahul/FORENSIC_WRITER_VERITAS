Design a modern cybersecurity web application interface for a project called **Automated Digital Forensics Reporting Tool**.

The system is an AI-assisted digital investigation platform that converts raw forensic data (logs, metadata, device activity records) into structured, court-ready forensic reports.

The interface should look like a **professional forensic investigation dashboard used by cybersecurity analysts and law enforcement agencies**.

Design style:
• Dark cybersecurity theme
• Deep navy / midnight blue gradient background
• Neon blue highlights and glow effects
• Glassmorphism UI cards
• Minimal futuristic typography
• Clean dashboard layout
• Professional and trustworthy visual design

The UI should include the following sections:

1. HERO SECTION
   Headline: “Automated Digital Forensics & Investigation Reporting”

Subtext explaining:
The platform processes digital evidence, extracts logs and metadata, verifies integrity using SHA-256 hashing, and generates standardized forensic reports using AI.

Buttons:
• Start Investigation
• Learn More

Include subtle cyber grid background and glowing UI elements.

---

2. PROFESSIONAL GRADE TOOLS

Feature cards explaining core capabilities:

• SHA-256 Evidence Integrity
Automatically calculates file hashes to ensure evidence authenticity and court admissibility.

• AI Report Generation
Uses AI to convert processed forensic data into human-readable investigation reports.

• Secure Evidence Storage
Encrypted storage of forensic evidence and metadata.

• Neutral Investigation Reports
AI generates standardized reports with neutral tone and no assumptions.

Cards should be glassmorphism style with glowing borders.

---

3. ROLE BASED ACCESS SYSTEM

Create a horizontal storytelling section explaining the stakeholders in the system.

Panel 1: System Administrator
Manages users, assigns roles, monitors activity, maintains system security.

Panel 2: Forensic Investigator
Creates cases, uploads evidence, analyzes extracted metadata, generates investigation reports.

Panel 3: Legal Authority / Court Reviewer
Reviews final reports, verifies evidence authenticity, downloads investigation reports.

Each panel should look like a futuristic dashboard preview.

---

4. INVESTIGATION WORKFLOW

Create a visual step-by-step forensic investigation pipeline.

Steps:

1. Upload Evidence
2. SHA-256 Hash Validation
3. Evidence Parsing and Metadata Extraction
4. AI Report Generation
5. Download Forensic Report

Design this as a vertical timeline with glowing step indicators.
